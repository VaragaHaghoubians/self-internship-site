AI/ML Roadmap Coverage for This Self-Internship

This file maps the external Comprehensive AI/ML Learning & Research Roadmap into this repository so the internship explicitly covers the same domains and stages.

Companion source mapping:

  • ROADMAP_SOURCES.md

What changed

The original self-internship already covered the engineering core well:

  • Software engineering
  • Production ML / MLOps
  • Full-stack delivery
  • IoT and industrial data
  • Capstone integration
  • NLP, RAG, and agentic AI

The external roadmap adds several advanced and expert-level requirements that were not explicit in the original 12-month plan:

  • Mathematical foundations as a continuous track
  • LLM fine-tuning and formal evaluation
  • Model quantization and efficient inference
  • Advanced agent reliability, observability, and evaluation
  • Monitoring, drift detection, SLOs, and safe rollout patterns
  • Research methodology, paper reproduction, and open-source contribution
  • Large-scale systems and expert-level portfolio work

This repository now treats the internship as:

  • Months 1-12: core engineering internship
  • Months 13-18: advanced AI systems track
  • Months 19-24: research and expert track

Continuous tracks that must run across all phases

  • Math refresh: linear algebra, calculus, probability, statistics, information theory
  • Paper reading: at least one paper or technical deep dive each week
  • Writing: one technical note, blog draft, or postmortem each month
  • Evaluation discipline: define success metrics before building
  • Portfolio discipline: every major month should end with a demo artifact and README

Gaps that were implicit before and are now explicit

  • Fine-tuning beyond prompting

How to use this file

  1. Treat the original 12-month roadmap as the core engineering internship.
  2. Treat Phase 7 and Phase 8 as required if the goal is alignment with the full external AI/ML roadmap.
  3. Use ROADMAP_SOURCES.md to choose the exact books, papers, tools, and courses for that phase.
  4. When updating any monthly plan, check whether it advances at least one domain and one stage competency from the external roadmap.
