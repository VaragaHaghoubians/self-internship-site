How to use this file

  1. Start with the phase you are currently in.
  2. Pull only the sources relevant to that month instead of trying to read everything at once.
  3. For advanced and expert months, combine one book or course, one paper, and one implementation tool.
  4. When you update a monthly plan, add the exact sources you actually used.
