8-Week Emergency Plan

Month 1

  • Week 1: Python fundamentals, Git basics, Linux basics
  • Week 2: NumPy, Pandas, visualization, SQL basics
  • Week 3: scikit-learn workflow and machine learning foundations
  • Week 4: PyTorch basics, MongoDB basics, integration project

Month 2

  • Week 5: Java syntax and control flow
  • Week 6: Java OOP and class design

Weekly Operating Rules

  1. Study one main topic at a time.
  2. Code every day.
  3. Update progress/mistakes.md whenever something breaks.
  4. Update progress/daily-log.md at the end of the day.
  5. Commit at least once per day.
  6. Do not move to the next week if the current week's deliverables are incomplete.
