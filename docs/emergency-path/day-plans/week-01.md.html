Week 1: Python Fundamentals, Git, Linux

Goal

Build enough Python fluency to write simple scripts without copying from tutorials.

Daily Time Split

  • 90 min Python learning
  • 60 min exercises
  • 45 min Git or Linux practice
  • 45 min review and logging

Day 1

  • Variables
  • Data types
  • Operators
  • Input and output
  • if, elif, else
  • Write variables_and_types.py
  • Write conditions.py

Day 2

  • for loops
  • while loops
  • range
  • Functions
  • Parameters and return values
  • Lists
  • Solve 8 to 10 small problems

Day 3

  • Strings
  • Indexing and slicing
  • Dictionaries
  • Sets
  • Basic string methods
  • Build a small text-processing script

Day 4

  • File reading
  • File writing
  • Exceptions with try and except
  • Modules and imports
  • Build a script that reads a text file and reports line count and word count

Day 5

  • Function practice
  • Break a larger task into smaller functions
  • Start mini project: command-line contact book or calculator
  • Learn Git commands: git init, git status, git add, git commit, git log

Day 6

  • Basic classes and objects
  • __init__
  • Instance attributes
  • Methods
  • Finish the mini project
  • Refactor repeated logic into functions or a simple class

Day 7

  • Linux basics: pwd, ls, cd, mkdir, cp, mv, rm, cat, grep, find
  • Git basics: branch, checkout, merge
  • Review weak Python topics
  • Push your code if you have a remote repo

Deliverables

  • At least 10 Python exercise files
  • 1 mini CLI project
  • 1 Git repo with daily commits
  • Updated mistakes.md and daily-log.md

Pass Check

You are ready for Week 2 only if you can:

  • write a function without looking up syntax
  • loop through a list and dictionary
  • read and write a file
  • explain the difference between a list and a dictionary
  • make a Git commit confidently
