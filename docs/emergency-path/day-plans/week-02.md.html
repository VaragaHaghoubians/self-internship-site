Week 2: NumPy, Pandas, Visualization, SQL

Goal

Learn how to load, inspect, clean, and summarize tabular data.

Daily Time Split

  • 90 min NumPy or Pandas
  • 60 min hands-on data work
  • 45 min SQL
  • 45 min review and notes

Day 1

  • NumPy arrays
  • Shapes and dimensions
  • Indexing and slicing
  • reshape
  • mean, sum, std
  • Practice with small arrays in a notebook

Day 2

  • Broadcasting
  • Element-wise operations
  • Random number generation
  • Dot product
  • Convert Python lists to NumPy arrays
  • Solve 5 array manipulation tasks

Day 3

  • Pandas Series and DataFrame
  • read_csv
  • head, tail, info, describe
  • Column selection
  • Row filtering
  • Load a real CSV dataset

Day 4

  • Missing values
  • fillna, dropna
  • Sorting
  • Grouping with groupby
  • Aggregations
  • Clean your dataset and save a new CSV

Day 5

  • merge
  • Basic feature creation
  • Matplotlib line, bar, histogram, scatter
  • Seaborn heatmap and boxplot
  • Build one notebook with 5 useful charts

Day 6

  • SQL SELECT
  • WHERE
  • ORDER BY
  • LIMIT
  • COUNT, SUM, AVG
  • Solve 10 SQL exercises

Day 7

  • SQL GROUP BY
  • HAVING
  • INNER JOIN
  • LEFT JOIN
  • Review NumPy and Pandas
  • Solve 10 more SQL exercises

Deliverables

  • 1 cleaned dataset
  • 1 exploratory data analysis notebook
  • At least 20 solved SQL queries
  • 1 short summary of findings in markdown

Pass Check

You are ready for Week 3 only if you can:

  • explain rows vs columns in a DataFrame
  • filter data with Pandas
  • group and aggregate data
  • create basic charts from a dataset
  • write a SQL query with JOIN and GROUP BY
