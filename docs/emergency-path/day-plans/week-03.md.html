Week 3: Machine Learning with scikit-learn

Goal

Understand the standard ML workflow from dataset to evaluation.

Daily Time Split

  • 90 min ML concepts and code
  • 75 min notebook practice
  • 30 min SQL or Git review
  • 45 min mistakes log and recap

Day 1

  • Supervised vs unsupervised learning
  • Features and targets
  • Train/test split
  • Regression vs classification
  • Load a small dataset and split it correctly

Day 2

  • Linear regression
  • Prediction
  • Mean absolute error
  • Mean squared error
  • R^2
  • Build a simple regression notebook

Day 3

  • Logistic regression
  • Classification labels
  • Accuracy
  • Precision
  • Recall
  • Confusion matrix
  • Build a binary classification notebook

Day 4

  • Decision trees
  • Random forest
  • K-nearest neighbors
  • Compare 3 models on one dataset
  • Write down why one performs better

Day 5

  • Preprocessing
  • StandardScaler
  • OneHotEncoder
  • Pipeline
  • ColumnTransformer
  • Avoid data leakage

Day 6

  • Cross-validation
  • Hyperparameter basics
  • Overfitting vs underfitting
  • Feature importance intuition
  • Improve yesterday's pipeline

Day 7

  • Review weak ML concepts
  • Finalize one regression or classification mini project
  • Write a README explaining:
  • problem
  • data
  • preprocessing
  • model
  • evaluation

Deliverables

  • 2 ML notebooks
  • 1 model comparison table
  • 1 pipeline using preprocessing and a model
  • 1 project README

Pass Check

You are ready for Week 4 only if you can:

  • explain train/test split
  • explain overfitting
  • train and evaluate a scikit-learn model
  • use a preprocessing pipeline
  • choose appropriate metrics for regression or classification
