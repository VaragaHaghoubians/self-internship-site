Week 4: PyTorch Basics, MongoDB Basics, Integration

Goal

Get a practical introduction to deep learning and understand MongoDB CRUD.

Daily Time Split

  • 90 min PyTorch or neural network concepts
  • 60 min coding
  • 45 min MongoDB
  • 45 min review and cleanup

Day 1

  • What a neural network is
  • Tensors
  • Inputs, weights, outputs
  • Forward pass intuition
  • Activation functions: ReLU, sigmoid, softmax
  • Create and inspect tensors in PyTorch

Day 2

  • Loss functions
  • Gradient descent
  • Epochs
  • Batch size
  • Learning rate
  • Build a tiny training loop on dummy data

Day 3

  • torch.nn
  • Dataset
  • DataLoader
  • Training and evaluation loop
  • Train a simple model on tabular or toy data

Day 4

  • MNIST or another beginner dataset
  • Simple fully connected network
  • Evaluate predictions
  • Save the notebook and explain the results

Day 5

  • MongoDB concepts
  • Database vs collection vs document
  • CRUD operations
  • insertOne, find, updateOne, deleteOne
  • Practice in mongosh or MongoDB Compass if installed

Day 6

  • PyMongo connection
  • Insert and query documents from Python
  • Compare SQL tables vs MongoDB documents
  • Build one Python CRUD script

Day 7

  • Consolidation day
  • Finish one small final Month 1 project:
  • option 1: Pandas + SQL analysis
  • option 2: scikit-learn project with README
  • option 3: PyTorch notebook plus MongoDB logging script
  • Review all weak topics from Month 1

Deliverables

  • 1 PyTorch notebook
  • 1 MongoDB CRUD script
  • 1 final Month 1 project
  • updated Git history and logs

Pass Check

You are ready for Month 2 only if you can:

  • explain what a tensor is
  • describe a basic training loop
  • insert and query documents in MongoDB
  • compare SQL and MongoDB at a basic level
  • present one finished project from Month 1
