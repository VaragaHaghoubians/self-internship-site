Day 1

Create these files in practice/python/day-01/:

  • variables_and_types.py
  • conditions.py

Solve these tasks:

  1. Store your name, age, and target role in variables and print them.
  2. Convert "25" to an integer and add 5.
  3. Read one number and print whether it is positive, negative, or zero.
  4. Read two numbers and print the larger one.
  5. Print numbers from 1 to 20, and print "even" next to the even ones.

How To Work

For each topic:

  1. Read for 20 to 30 minutes.
  2. Code for 60 to 90 minutes.
  3. Write down what failed.
  4. Commit your work.

End-of-Day Checklist

  • Code written
  • Errors recorded
  • One short summary added to progress/daily-log.md
  • Changes committed with Git
