🚀 My Learning Journey: From Beginner to Data Scientist + IoT Engineer

Welcome! This folder is your entire roadmap broken into small, doable steps. Every subfolder = one month. Every week inside has its own folder with:

  • A README.md that explains WHAT to do and WHY
  • A .py Python file with code examples and hints in the comments

THE TWO TYPES OF FILES — AND HOW THEY WORK TOGETHER

knowledge-base/
│
├── learning-journey/          ← YOUR WEEKLY WORK PROGRAM
│   └── (this folder)          Read these in strict order, one week at a time.
│                              Each file has tasks you must complete.
│
└── zero-to-hero-guides/       ← REFERENCE TEXTBOOKS
    └── python/, sql/, etc.    Read the relevant guide BEFORE starting the
                               learning-journey week that uses that tool.
                               Keep them open as a reference while working.

Learning journey = what you DO each week (applied, project-driven). Zero-to-hero guides = what you READ to prepare (deep dives, cheatsheets).

They are not redundant — they work together. The table below shows exactly when.
