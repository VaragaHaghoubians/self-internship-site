🟡 Month 2: Core Data Science & Hardware Intro

Theme: Analyze real data and collect your own sensor readings. Pre-requisite: You can use the terminal, push to GitHub, and open a Jupyter Notebook.

Why This Month Matters

As a master's student in data science, this is your core skill set. By the end of this month you will:

  • Load, clean, and explore any dataset
  • Query a database with SQL
  • Fetch data from the internet automatically
  • Collect your own data using a real sensor!

Week-by-Week Plan

Week 5 — Pandas, NumPy, Visualization & Statistics

📁 Folder: week-05-pandas-numpy/ 📄 Files: pandas_basics.py, numpy_basics.py, visualization_basics.py, statistics_basics.py

Pandas is the #1 tool for data manipulation in Python. NumPy is the math engine underneath Pandas.

You will learn:

  • How to load a CSV file into a DataFrame
  • How to filter rows (e.g., "show me only rows where temperature > 25")
  • How to calculate mean, max, min, count
  • How to sort and group data
  • How to create charts: histograms, bar charts, scatter plots, heatmaps
  • Descriptive statistics, hypothesis testing (t-test, chi-square), correlation
  • NumPy arrays: slicing, broadcasting, reshaping, random numbers

Week 6 — Data Cleaning

📁 Folder: week-06-data-cleaning/ 📄 File: data_cleaning.py

Real data is MESSY. Missing values, wrong formats, duplicates. Your job as a data scientist is to clean it before analysis.

Week 6 (extra) — SQL Basics

📁 Folder: week-06-sql/ 📄 File: sql_basics.py

SQL (Structured Query Language) lets you query databases — the same skill asked in almost every data job interview. This file covers SELECT, WHERE, GROUP BY, JOIN, subqueries, and CASE WHEN using Python's built-in sqlite3.

Week 7 — APIs & Automation

📁 Folder: week-07-apis/ 📄 File: fetch_api_data.py

APIs (Application Programming Interfaces) let your Python code fetch live data from the internet — weather, stock prices, news, etc.

Week 8 — Arduino Hardware

📁 Folder: week-08-arduino/ 📄 File: read_serial_data.py

This week you connect your ELEGOO Arduino kit to Python! You'll collect REAL temperature data and save it to a CSV. This becomes the dataset for your Portfolio Project 1.

🏆 Portfolio Project 1

Name: Sensor Data EDA (Exploratory Data Analysis) What you do:

  1. Collect temperature data from your Arduino (Week 8)
  2. Load it into Pandas
  3. Clean any missing or bad readings
  4. Create 5 charts showing temperature trends
  5. Write a README explaining 3 findings
  6. Push to GitHub as a new repo: sensor-eda-analysis

✅ End of Month 2 Checklist

  • [ ] Can load a CSV and calculate basic stats with Pandas
  • [ ] Can handle missing values (dropna, fillna)
  • [ ] Can write basic SQL queries (SELECT, WHERE, GROUP BY)
  • [ ] Can fetch data from a URL with Python's requests library
  • [ ] Have Arduino blinking + reading temperature sensor
  • [ ] Portfolio Project 1 pushed to GitHub
