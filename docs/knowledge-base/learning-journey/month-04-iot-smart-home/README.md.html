🔴 Month 4: The Smart Home IoT Integration

Theme: Combine hardware + Python to build a real IoT system. Pre-requisite: Finished Months 1-3. Arduino kit arrived. ESP32 boards arrived.

Why This Month Makes You Unique

Most data scientists only work with pre-made datasets. You will have a system that continuously generates its OWN data from real sensors.

The complete pipeline you'll build:

[ESP32 Sensor] --WiFi--> [Flask Server on your laptop] --saves to--> [SQLite DB]
                                                                           ↓
                                                              [Live Chart in Browser]

This is a real IoT data pipeline — the same concept used in smart factories, hospitals, weather networks, and smart cities.

Week-by-Week Plan

Week 13 — ESP32 Setup

📁 Folder: week-13-esp32-setup/

  • Install ESP32 board support in Arduino IDE
  • Upload a "Hello World" WiFi sketch to the ESP32
  • Verify it connects to your home router

Week 14 — Flask API Server

📁 Folder: week-14-flask-api/ 📄 File: flask_server.py

  • Build a simple Python web server with Flask
  • Create an endpoint that accepts POST requests from the ESP32
  • Test it using curl from your terminal

Week 15 — Wireless Data Transmission

📁 Folder: week-15-wireless-data/ 📄 File: esp32_sensor.ino (Arduino code for ESP32)

  • Wire DHT11 to ESP32
  • Program ESP32 to read temperature and POST to Flask

Week 16 — Live Dashboard

📁 Folder: week-16-live-dashboard/ 📄 File: dashboard_app.py

  • Save incoming data to SQLite database
  • Serve a live HTML chart in the browser (auto-refreshes)

🏆 Portfolio Project 3

Name: Live Smart Home Dashboard GitHub repo: smart-home-dashboard
