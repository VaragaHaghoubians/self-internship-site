Week 13: ESP32 Setup Guide

What is ESP32 and Why Switch from Arduino UNO?

FeatureArduino UNOESP32
WiFi❌ No✅ Built-in
Bluetooth❌ No✅ Built-in
Speed16 MHz240 MHz
RAM2 KB520 KB
Price~$25~$5
ProgrammingArduino C++Arduino C++ (same!)

The ESP32 speaks the same language as Arduino UNO — so all your Arduino code from Month 2 can be copied and adapted for ESP32.

Step 1: Add ESP32 Support to Arduino IDE

The Arduino IDE doesn't know about ESP32 by default. We need to add it.

  1. Open Arduino IDE
  2. Go to: File → Preferences
  3. Find the "Additional Boards Manager URLs" field
  4. Paste this URL:
   https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json
  1. Click OK
  1. Go to: Tools → Board → Boards Manager
  2. Search for: esp32
  3. Find "esp32 by Espressif Systems" and click Install
  4. Wait for the download to finish (may take a few minutes)

Step 2: Select Your ESP32 Board

After installation:

  • Tools → Board → esp32 → ESP32 Dev Module
  • Tools → Port → COMx (Windows) or /dev/ttyUSB0 (Linux)
  • Plug in your ESP32 via USB first
  • If no port appears: Install the CP2102 driver from:

https://www.silabs.com/developers/usb-to-uart-bridge-vcp-drivers

Step 3: Upload a Test Sketch

Create a new sketch in Arduino IDE and paste this code:

// ESP32 WiFi Connection Test
// This sketch connects to your WiFi and blinks the built-in LED
// when connected

#include <WiFi.h>

// ⚠️ CHANGE THESE TO YOUR OWN WIFI CREDENTIALS
const char* ssid     = "YOUR_WIFI_NAME";
const char* password = "YOUR_WIFI_PASSWORD";

void setup() {
    Serial.begin(115200);
    pinMode(2, OUTPUT);       // Built-in LED on most ESP32 boards is pin 2

    Serial.println("Connecting to WiFi...");
    WiFi.begin(ssid, password);

    // Wait until connected
    while (WiFi.status() != WL_CONNECTED) {
        delay(500);
        Serial.print(".");
    }

    Serial.println("\nConnected to WiFi!");
    Serial.print("My IP address: ");
    Serial.println(WiFi.localIP());
    // IMPORTANT: Note down this IP address!
    // Your Flask server needs to know this IP to receive data from ESP32.
}

void loop() {
    // Blink the LED to show we're connected and running
    digitalWrite(2, HIGH);
    delay(1000);
    digitalWrite(2, LOW);
    delay(1000);
}

Upload it and open Tools → Serial Monitor (set baud rate to 115200). You should see "Connected to WiFi!" and an IP address like 192.168.1.42.

Step 4: Find Your Laptop's IP Address

Your ESP32 will need to send data to your laptop's IP address.

On Windows:

ipconfig

Look for "IPv4 Address" under your WiFi adapter. Example: 192.168.1.10

On Linux:

hostname -I

Write down your laptop's IP. You'll need it in Week 15.
