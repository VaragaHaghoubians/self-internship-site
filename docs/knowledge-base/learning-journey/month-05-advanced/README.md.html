🟣 Month 5+: The "Unicorn" Expansion (Optional)

⛔ STOP: Only open this folder after you have:

  • Finished Months 1-4
  • Pushed 3 portfolio projects to GitHub
  • Started applying for Junior Data Analyst / Data Scientist jobs

You are already competitive after Month 3. Do not skip the job search!

The "Unicorn" Stack

By adding these skills on top of your data science + IoT foundation, you become a rare profile that almost no one else has.

🔵 Track A: Django (Full-Stack Web)

What it is: Django is a full web framework. Instead of the simple Flask servers you built in Month 4, Django handles user logins, admin panels, complex databases, and is used in production at Instagram, Pinterest, and Spotify.

Learn in this order:

  1. Django Girls Tutorial — 1 weekend, free
  2. MDN Django Guide — deeper dive, free
  3. Upgrade your Smart Home Flask server to a full Django app with:
  • User accounts (login/logout)
  • PostgreSQL instead of SQLite
  • Admin panel to manage sensor data

Project: Full-stack Django app — e.g., a Job Board or Smart Home Management Portal.

🔵 Track B: Java & Big Data (Enterprise)

What it is: Java is the language of enterprise software. PySpark lets Python handle datasets too large for Pandas (millions of rows).

Learn in this order:

  1. University of Helsinki Java MOOC — free, takes ~2 months
  2. PySpark: pip install pyspark, then follow DataCamp's PySpark tutorial
  3. Build a pipeline:
  • Java app: clean a large CSV
  • PySpark: aggregate it into statistics

🔵 Track C: Cloud & MLOps (AWS)

What it is: Deploy your Dockerized ML apps to the internet so anyone in the world can access them — not just people on your WiFi.

Learn in this order:

  1. Create a FREE AWS account: https://aws.amazon.com/free/
  2. Launch an EC2 instance (a Linux server in the cloud)
  3. SSH into it: ssh -i your-key.pem ubuntu@your-ec2-ip
  4. Install Docker on EC2
  5. Push your docker image to DockerHub, pull it on EC2, run it

Your dashboard URL changes from http://localhost:5000 to http://your-ec2-ip:5000 — accessible from any phone or computer in the world!

🔵 Track D: Deep Learning (PyTorch)

What it is: Neural networks that can recognize images, understand text, and achieve human-level performance on complex tasks.

Guided practice file: deep_learning_basics.py (in this folder)

  • Trains a real neural network on the Breast Cancer dataset
  • Covers: tensors, forward pass, loss, backprop, training loop, evaluation
  • Includes a full cheatsheet of PyTorch patterns
  • 5 YOUR TASKS including: experiment with architecture, add early stopping,

convert to multi-class (Iris), train on your own IoT sensor data

After the guided file, go deeper:

  1. fast.ai course — free, best top-down deep learning course
  2. Build an image classifier: can your model tell the difference between

your room being clean vs. messy? (use photos from your phone)

  1. Serve the model via Django and deploy to AWS

� Track E: STM32 (Professional Embedded Systems)

What it is: STM32 is the microcontroller used in real industrial products — medical devices, automotive electronics, PLCs, robots, and aerospace systems. It bridges your Python/ML skills with professional hardware engineering.

When to take this track: After month-04 IoT is complete. You already know Arduino C++ from week-08 and ESP32 from month-04 — the syntax is identical. STM32 adds: 12-bit ADC precision, hardware interrupt system, real-time OS (FreeRTOS), and a professional debugger (ST-Link).

Guided practice file: stm32_guide.py (in this folder)

  • Where to buy and which board (Nucleo-F103RB recommended, ~$12)
  • Arduino IDE vs. STM32CubeIDE — which to use when
  • 4 progressively advanced programs (LED, ADC sensor, UART to Python, interrupts)
  • Full industrial sensor logger project connecting STM32 → Python → Flask → dashboard
  • 4-week learning path with free resources
  • 5 YOUR TASKS including connecting alarms to your existing Flask server

Why it matters for your career:

  • ESP32 on a CV = "I do hobby IoT"
  • STM32 on a CV = "I work with industrial embedded systems"
  • STM32 + Python/ML = the exact profile companies look for in

Industrial Engineering, Mechatronics, and Industry 4.0 roles
