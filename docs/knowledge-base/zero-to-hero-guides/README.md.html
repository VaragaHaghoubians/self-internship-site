🛠️ Tools You Need to Install

Python

  • Download: https://www.python.org/downloads/
  • IDE: VS Code + Python extension OR PyCharm Community
  • Run files: python filename.py

SQL

  • Beginner (no install needed): SQLite — built into Python, use VS Code + "SQLite Viewer" extension
  • Advanced: MySQL Community Server or PostgreSQL
  • GUI tool: DBeaver (free, works with all three)

Git

  • Download: https://git-scm.com/downloads
  • Run: git --version to verify

Linux

  • Windows users: Install WSL2 (Windows Subsystem for Linux)
  • Run in PowerShell as Admin: wsl --install
  • OR use Git Bash (comes with Git)

Docker

  • Download Docker Desktop: https://www.docker.com/products/docker-desktop
  • Verify: docker --version | docker compose version
  • No extra installs needed — Docker pulls everything automatically

Java

  • Download JDK 21: https://adoptium.net/
  • Verify: java --version
  • Compile: javac MyFile.java | Run: java MyFile

Arduino

  • Download Arduino IDE: https://www.arduino.cc/en/software
  • Board: Arduino Uno or Nano

Django

  • Install after Python: pip install django

IoT / Raspberry Pi

  • Install Raspberry Pi OS (if using Pi)
  • Python libraries: pip install paho-mqtt RPi.GPIO
