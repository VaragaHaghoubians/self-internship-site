Arduino — Zero to Hero

Prerequisite: Basic programming concepts (any language). IoT concepts help.

What You Will Learn

  • What Arduino is and how it differs from Raspberry Pi
  • Arduino IDE setup and uploading sketches
  • Digital output — blinking LEDs
  • Digital input — reading buttons
  • Analog input — potentiometers and sensors
  • PWM — dimming LEDs and controlling servos
  • Serial communication — talking to your PC
  • Real sensors: DHT11, ultrasonic HC-SR04
  • I2C LCD display
  • Capstone: weather station

Install

  1. Download Arduino IDE: https://www.arduino.cc/en/software
  2. Connect your Arduino board via USB
  3. Select board: Tools → Board → Arduino Uno (or your board)
  4. Select port: Tools → Port → COMx (Windows) or /dev/ttyUSBx (Linux)

How to Use

  1. Open arduino_zero_to_hero.ino in Arduino IDE
  2. Read the // TEACHER SAYS: comments for each day
  3. Wire up the circuit as described
  4. Upload the sketch for that day: Sketch → Upload (Ctrl+U)
  5. Open Serial Monitor (Tools → Serial Monitor) to see output
