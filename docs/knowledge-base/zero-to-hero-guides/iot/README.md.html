IoT — Zero to Hero

Prerequisite: Complete Python first. Django and Docker knowledge is helpful.

What You Will Learn

  • IoT architecture: sensors, actuators, protocols
  • Raspberry Pi GPIO programming
  • Reading DHT11/DHT22 temperature & humidity sensors
  • MQTT protocol — publish and subscribe
  • Logging sensor data to CSV and SQLite
  • Building a REST API for IoT data with Flask
  • Live dashboard with Matplotlib

Install

pip install paho-mqtt requests flask matplotlib
# On Raspberry Pi only:
pip install RPi.GPIO adafruit-circuitpython-dht

Hardware (Optional but Recommended)

  • Raspberry Pi 4 (or 3B+)
  • DHT22 temperature/humidity sensor
  • LEDs, resistors, jumper wires
  • Breadboard
All examples include a simulation mode — you can run every lesson on your PC without any hardware.

IoT — Zero to Hero

Connect the physical world to software with Python and Raspberry Pi.

Hardware (Recommended)

BoardCostBest For
Raspberry Pi 4~$35Full Linux, Python, runs this file directly
ESP32~$5WiFi/Bluetooth built-in, MicroPython
Arduino Uno~$10Simple hardware control (see arduino/ folder)
No hardware? The file runs in simulation mode on any PC — all sensor data is simulated.

Install

pip install paho-mqtt adafruit-circuitpython-dht flask requests matplotlib

On Raspberry Pi also install:

pip install RPi.GPIO
sudo apt install python3-dev
