Python — Zero to Hero

What is Python?

Python is the #1 language for Data Science. It's readable, powerful, and has libraries for everything.

Install Python

  1. Go to: https://www.python.org/downloads/
  2. Download Python 3.11 or newer
  3. ✅ Check "Add Python to PATH" during install
  4. Verify in terminal: python --version

Install VS Code (recommended IDE)

  1. Go to: https://code.visualstudio.com/
  2. Install the "Python" extension by Microsoft
  3. Open the .py file → press F5 to run

Run the learning file

python python_zero_to_hero.py

Libraries you will need (install later)

pip install numpy pandas matplotlib seaborn scikit-learn jupyter
