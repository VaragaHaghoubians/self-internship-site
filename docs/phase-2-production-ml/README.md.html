Phase 2: Production Machine Learning (MLOps)

Duration: Month 3–4 Core Goal: Learn how models are deployed and maintained in the real world.

By the end of this phase you can package any Python application into a Docker container, track ML experiments systematically, spin up a Linux cloud server, and set up automated testing with CI/CD.

Month 3 — Containerization & MLOps Basics

📁 month-03-docker-mlops/

Skills to master:

  • Docker: Dockerfile, docker build, docker run, port binding, volumes
  • Docker Compose: multi-service applications
  • MLflow: tracking experiments, logging metrics/params, registering models
  • Model versioning and reproducibility

Deliverable: Take your Month 2 FastAPI ML app and run it inside a Docker container. Anyone should be able to docker run it and get predictions.

Reference Project: projects/foundations/project-3-house-price-pipeline/

Month 4 — Cloud Computing & CI/CD

📁 month-04-cloud-cicd/

Skills to master:

  • AWS EC2 (or GCP): launching a Linux server, SSH connection, security groups
  • Basic Linux server admin: installing packages, running services
  • GitHub Actions: writing a .github/workflows/ YAML file to test on every push
  • Deploying a Docker container to a remote server

Deliverable: Deploy your Dockerized ML model to a free-tier cloud server so anyone on the internet can hit your /predict API endpoint.

Reference Project: projects/foundations/project-4-news-classification/
