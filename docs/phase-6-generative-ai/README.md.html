Phase 6: Generative AI & Agentic Systems

Duration: Month 10–12 Core Goal: Put AI in the driver's seat — build systems where LLMs use tools, reason over documents, and take autonomous actions.

This phase is unlocked after the 9-month program. By now you have FastAPI, Docker, cloud deployment, Django, async tasks, and SQL — the exact infrastructure that production AI systems are built on.

Month 10 — NLP Foundations & LLM Basics

📁 month-10-nlp-llm-basics/

Skills to master:

  • Classical NLP: tokenization, TF-IDF, text classification (scikit-learn + NLTK)
  • What LLMs actually are: transformers, tokens, embeddings, context windows
  • Prompt engineering: zero-shot, few-shot, chain-of-thought, structured output
  • Calling LLM APIs: OpenAI GPT-4o, Hugging Face Transformers (local models)
  • Wrapping an LLM call in a FastAPI endpoint

Deliverable: A FastAPI endpoint that accepts a user question and returns an LLM-generated answer. Deployed in Docker on your existing cloud server.

Resources: month-10-nlp-llm-basics/resources/

  • nlp_and_llm_basics.py — zero-to-hero NLP + LLM guide (8 parts)
  • news_classification_nlp.py — real NLP classification project (reference)

Install:

pip install scikit-learn nltk transformers datasets openai fastapi uvicorn

Month 11 — RAG — Retrieval-Augmented Generation

📁 month-11-rag-vector-databases/

Skills to master:

  • Why LLMs hallucinate and what RAG solves
  • Vector embeddings: converting text to high-dimensional vectors
  • Vector databases: ChromaDB (local), pgvector (PostgreSQL), Pinecone (cloud)
  • The RAG pipeline: chunk documents → embed → store → retrieve → inject → generate
  • LangChain: document loaders, text splitters, retrievers, RetrievalQA chains
  • Integrating RAG into Django: users upload documents, chat with them via LLM

Deliverable: A "Chat with your PDF" web app — user uploads a PDF, the system indexes it into ChromaDB, and the user can ask questions answered from the document's actual content.

Resources: month-11-rag-vector-databases/resources/

  • rag_pipeline.py — zero-to-hero RAG guide (7 parts + FastAPI app)

Install:

pip install chromadb openai langchain langchain-openai langchain-community pypdf sentence-transformers tiktoken python-dotenv

Month 12 — Agentic AI — Autonomous Systems

📁 month-12-agentic-ai/

Skills to master:

  • The ReAct loop: how agents Reason → Act → Observe → repeat
  • OpenAI tool/function calling: structured JSON tool invocation
  • Building an agent loop from scratch (to understand what frameworks abstract)
  • LangChain agents: create_tool_calling_agent, AgentExecutor
  • LangGraph: stateful, multi-step agents with conditional branching
  • Agent memory: short-term (conversation), long-term (vector DB), episodic (SQL log)
  • Multi-agent systems: supervisor + specialist agents running in parallel
  • Production safety: max-step limits, cost tracking, input validation, timeouts

Deliverable: An autonomous AI agent that can:

  1. Query your SQL database via a tool
  2. Search the web via a tool
  3. Run a Python data analysis script via a tool
  4. Summarize findings — all from a single natural-language instruction

Resources: month-12-agentic-ai/resources/

  • agentic_ai.py — zero-to-hero agent guide (9 parts)

Install:

pip install openai langchain langgraph langchain-openai python-dotenv duckduckgo-search
