INSTRUCTIONS — Project 2: Data Analyst

Retail Sales Performance Analysis

Your Role

You are acting as a Data Analyst. In industry, a Data Analyst is expected to:

  • Clean and prepare raw data for analysis
  • Aggregate and summarize data to answer business questions
  • Create clear, informative visualizations
  • Identify trends, outliers, and anomalies
  • Write concise reports with actionable findings

Data Analysts do NOT necessarily build predictive models — the focus is on describing what happened and why, not predicting what will happen.

What to Study Before Coding

Python & Pandas:

  • [ ] How to read CSV files with pd.read_csv()
  • [ ] How to parse dates: pd.read_csv(..., parse_dates=['Order Date'])
  • [ ] How to use .groupby() + .agg() for summarizing data
  • Google: "pandas groupby agg tutorial"
  • [ ] How to create derived columns: df['Profit Margin'] = df['Profit'] / df['Sales']
  • [ ] How to use .sort_values() and .nlargest() / .nsmallest()
  • [ ] How to work with datetime objects in pandas:
  • Extracting year: df['Year'] = df['Order Date'].dt.year
  • Extracting month: df['Month'] = df['Order Date'].dt.month

Visualization:

  • [ ] Bar charts — good for comparisons across categories
  • [ ] Line charts — good for time series (sales over time)
  • [ ] Scatter plots — good for relationships between two numerical variables
  • [ ] Heatmaps — good for showing values across two dimensions
  • [ ] How to use plt.subplots() for multiple plots in one figure
  • [ ] How to customize labels, titles, colors

Business Concepts:

  • [ ] What is profit margin? → Profit / Sales × 100
  • [ ] What is a KPI (Key Performance Indicator)?
  • [ ] What does a negative profit mean for a product line?
  • [ ] What is a customer segment? How do businesses classify customers?
  • [ ] What is seasonality in sales data?

Phase-by-Phase Implementation Guide

Phase 1 — Setup (20 minutes)

  1. Create the folder structure:
   project_2_data_analyst/
       data/
       outputs/
           figures/
  1. Download the Superstore dataset from Kaggle (link in README.md)
  2. Rename the file to superstore.csv and place it in data/
  3. Install libraries:
   pip install pandas numpy matplotlib seaborn plotly

Phase 2 — Load and Explore Data (45 minutes)

What to do:

  1. Load the CSV with pd.read_csv(). Try adding encoding='latin1' if you get encoding errors.
  2. Parse date columns immediately: parse_dates=['Order Date', 'Ship Date']
  3. Check the dataset:
  • Shape, column names, data types
  • Missing values (.isnull().sum())
  • Data types — are dates loaded as datetime? (df.dtypes)
  • First/last rows: .head(), .tail()

Questions to answer in code:

  • How many unique orders are there? (Order ID)
  • How many unique customers?
  • How many product categories?
  • What date range does the data cover?
  • Is there any missing data?

Phase 3 — Data Cleaning & Feature Engineering (45 minutes)

Cleaning tasks:

  1. Check for missing values — are any important columns missing?
  • Drop or fill them depending on the column
  • For numerical columns, consider filling with median
  1. Parse date columns — if they weren't parsed automatically:
  • df['Order Date'] = pd.to_datetime(df['Order Date'])
  • df['Ship Date'] = pd.to_datetime(df['Ship Date'])
  1. Remove duplicate rows (if any):
  • Check: df.duplicated().sum()
  • Remove: df.drop_duplicates(inplace=True)

Feature Engineering — create these new columns:

New ColumnFormulaWhy
Profit MarginProfit / SalesProfitability ratio
Processing DaysShip Date - Order Date → .dt.daysOperational efficiency
YearOrder Date.dt.yearYear-level time series
MonthOrder Date.dt.monthMonthly trends
Month NameOrder Date.dt.month_name()Readable month labels
Year-MonthOrder Date.dt.to_period('M')Monthly grouping

Phase 4 — KPI Summary (30 minutes)

Before making charts, compute your top-level Key Performance Indicators. These are the numbers that give you the "big picture" at a glance.

Calculate these KPIs:

KPIHow to Compute
Total Revenuedf['Sales'].sum()
Total Profitdf['Profit'].sum()
Overall Profit Margindf['Profit'].sum() / df['Sales'].sum() * 100
Total Ordersdf['Order ID'].nunique()
Average Order Valuedf.groupby('Order ID')['Sales'].sum().mean()
Average Profit per Orderdf.groupby('Order ID')['Profit'].sum().mean()
Total Customersdf['Customer ID'].nunique()

Print these formatted as a KPI dashboard:

...

Phase 5 — Sales & Profit by Category (1 hour)

Analysis 1 — Category Level:

  • Group by Category, aggregate Sales and Profit (sum)
  • Compute profit margin per category
  • Create a grouped bar chart: Sales and Profit side by side for each category
  • Which category generates the most revenue? The most profit? What is the margin?

Analysis 2 — Sub-Category Level:

  • Group by Sub-Category, aggregate Sales and Profit (sum)
  • Sort by Profit (descending)
  • Create a horizontal bar chart showing profit per sub-category
  • Color negative bars RED, positive bars GREEN
  • This reveals "losing" product lines — critical business finding!

Key questions to answer:

  • What are the top 3 most profitable sub-categories?
  • What are the sub-categories losing money? Should the business discontinue them?
  • Is Furniture as a category profitable overall?

Phase 6 — Regional Analysis (1 hour)

Analysis 3 — Sales by Region:

  • Group by Region, aggregate Sales and Profit (sum)
  • Create a bar chart comparing regions
  • Which region has the most revenue? The highest margin?

Analysis 4 — Top 10 States by Sales:

  • Group by State, aggregate Sales (sum)
  • Sort descending, take top 10
  • Create a horizontal bar chart of top 10 states
  • Which state is the biggest market?

Analysis 5 — Category Performance by Region (Heatmap):

  • Create a pivot table: rows = Region, columns = Category, values = Profit
  • Use: df.pivot_table(values='Profit', index='Region', columns='Category', aggfunc='sum')
  • Plot as a heatmap: sns.heatmap(pivot, annot=True, fmt='.0f', cmap='RdYlGn')
  • This shows WHERE each category performs best and worst geographically

Phase 7 — Time Series Analysis (1 hour)

Analysis 6 — Monthly Sales Trend:

  • Group by Year-Month (or Year and Month), aggregate Sales (sum)
  • Plot a line chart: X-axis = time, Y-axis = monthly sales
  • Do you see any seasonal patterns? (e.g., spikes at end of year?)

Analysis 7 — Year-over-Year Growth:

  • Group by Year, compute total Sales and Profit
  • Compare 2014, 2015, 2016, 2017 (or whatever years are in your data)
  • Create a bar chart of annual revenue
  • Calculate YoY growth rate: (this_year - last_year) / last_year * 100

Analysis 8 — Monthly Sales Heatmap:

  • Create a pivot table: rows = Year, columns = Month, values = Sales
  • Plot as a heatmap
  • This visually shows which months are hot every year

Phase 8 — Customer Segment Analysis (45 minutes)

Analysis 9 — Revenue and Profit by Segment:

  • Group by Segment, aggregate Sales and Profit
  • Bar chart comparison
  • Which segment is most profitable? Which has the best margin?

Analysis 10 — Average Order Value by Segment:

  • Group by Customer ID and Segment to get per-customer totals
  • Then average by Segment
  • This shows average customer value across segments

Phase 9 — Discount Impact Analysis (45 minutes)

Analysis 11 — Discount vs. Profit (Scatter Plot):

  • Plot Discount (X) vs. Profit (Y) as a scatter plot
  • Color by Category
  • Add a trend line: use np.polyfit() or sns.regplot()
  • Does higher discount lead to lower profit? (usually yes!)

Analysis 12 — Average Profit by Discount Range:

  • Create discount bins: 0%, 1-20%, 21-40%, 41-60%, 60%+
  • Use: pd.cut(df['Discount'], bins=[0, 0.01, 0.2, 0.4, 0.6, 1.0], labels=[...])
  • Group by the bin, compute mean Profit
  • Bar chart: profit per discount range
  • This makes the discount-hurts-profit relationship very clear

Phase 10 — Shipping Analysis (30 minutes)

Analysis 13 — Average Processing Days by Ship Mode:

  • Group by Ship Mode, compute mean of Processing Days
  • Which ship mode is fastest? Slowest?
  • Bar chart
