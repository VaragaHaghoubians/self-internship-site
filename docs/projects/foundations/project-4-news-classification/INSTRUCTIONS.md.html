Phase 4 — Text Preprocessing (1.5 hours)

Write a function called clean_text(text) that:

  1. Converts to lowercase: text.lower()
  2. Removes HTML tags: use re.sub('<.*?>', '', text) with the re module
  3. Removes special characters and punctuation:

re.sub('[^a-zA-Z\s]', '', text) — keep only letters and spaces

  1. Tokenizes (splits into a list of words): text.split()
  2. Removes stop words:
   from nltk.corpus import stopwords
   stop_words = set(stopwords.words('english'))
   words = [w for w in words if w not in stop_words]
  1. Lemmatizes (converts words to their root form):
   from nltk.stem import WordNetLemmatizer
   lemmatizer = WordNetLemmatizer()
   words = [lemmatizer.lemmatize(w) for w in words]
  1. Rejoins words into a string: ' '.join(words)

Apply this function to all articles:

df['clean_text'] = df['text'].apply(clean_text)

Verify: Print an original vs. cleaned article side by side. Notice how the cleaned text is shorter and more focused.

Important note for BERT: Do NOT apply heavy preprocessing to the BERT version. BERT was trained on normal English text and its tokenizer handles capitalization and punctuation. For BERT, just use the raw text column.

Phase 5 — TF-IDF Baseline Model (2 hours)

Part 5.1 — Sampling: Training on 120,000 examples is slow. Use a sample:

# Take a stratified sample of 20,000 for training, 4,000 for testing
# This keeps class proportions the same as the full dataset

Use df.sample(n=..., random_state=42) or sklearn's train_test_split with stratify.

Part 5.2 — TF-IDF Vectorization:

from sklearn.feature_extraction.text import TfidfVectorizer

tfidf = TfidfVectorizer(
    max_features=50000,   # keep the 50,000 most common words
    ngram_range=(1, 2),   # include both single words AND pairs of words (bigrams)
    min_df=2,             # ignore words appearing in fewer than 2 documents
    max_df=0.95,          # ignore words appearing in more than 95% of documents
    sublinear_tf=True     # apply log normalization to term frequency
)

X_train_tfidf = tfidf.fit_transform(train_clean_texts)   # fit AND transform
X_test_tfidf  = tfidf.transform(test_clean_texts)        # transform ONLY

Note the same rule as Project 1: fit ONLY on training data!

Part 5.3 — Train Classifiers:

Model A — Logistic Regression:

from sklearn.linear_model import LogisticRegression
lr = LogisticRegression(C=5.0, max_iter=1000, random_state=42)
lr.fit(X_train_tfidf, y_train)
y_pred_lr = lr.predict(X_test_tfidf)

Model B — Linear SVM (usually best for text):

from sklearn.svm import LinearSVC
svm = LinearSVC(C=1.0, max_iter=2000, random_state=42)
svm.fit(X_train_tfidf, y_train)
y_pred_svm = svm.predict(X_test_tfidf)
